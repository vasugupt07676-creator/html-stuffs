CSS Tooltips

Zero-JavaScript tooltip system with four directions, six themes, and bouncy animations. Just add data attributes.

Directions

Tooltips can appear in all four directions using the data-tooltip-dir attribute.

Themes

Six built-in themes to match any UI. Mix and match freely.

Icon Buttons

Works perfectly with icon buttons for compact UI elements.
